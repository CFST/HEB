Welcome to the HTML Email Boilerplate site. This website and its sample code creates a template of sorts, absent of design or layout, that will help you avoid some of the major rendering problems with the most common email clients out there — Gmail, Outlook, Yahoo Mail, Lotus Notes, etc. While not a plug and play layout (ergo, you will have to do some work ;-), it will provide some helpful examples and snippets that will keep your design rendering as true-to-form as possible.

THE CODE

The meat of the project lies here in the Code section. Feel free to copy and paste as necessary. Click on the different code sections for more information on each snippet.

/** Hotmail header color reset:  Replace #000 (black) with your color. The !important is really what is overriding Hotmail's styling.  This is just a way to reset Hotmail to black.  If you have a specific color in mind for your headers than you still need to produce inline styles in that color for Gmail.  In fact, this will work the same inline with the !important declaration.  This way makes it clean and super easy for a tool to move the style inline.  If you need an inline styling tool, check out: http://premailer.dialect.ca/. 
**/
h1, h2, h3, h4 {
   color: #000 !important;
}






/** Yahoo link color reset (fix #1 of 2): This fix is from Brian Thies over at Campaign Monitor. Not uniform across clients. still have to set link color inline for gmail.  More info: http://www.campaignmonitor.com/blog/post/3226/two-solutions-to-yahoo-mail-turning-all-your-links-blue/ **/
a { color: #980; text-decoration: underline;} /** Yahoo Classic and other clients that support styling within the HEAD tags **/
.yshortcuts { color: #980 !important; text-decoration: underline !important;} /* Body text color for the New Yahoo */
.yshortcuts a span { color: #980 !important; text-decoration: underline !important; } /* Link text color for the New Yahoo */

/** BONUS: Adaptation of Brian Thies link color fix to render the Yahoo Short Cuts invisible.  You need to make the text color the same of the actual font color of your email and then set text-underline to none.**/
.yshortcuts { color: #000 !important; text-decoration: underline !important;} /* Body text color for the New Yahoo */
.yshortcuts a span { color: #000 !important; text-decoration: underline !important; } /* Link text color for the New Yahoo */

/** Yahoo paragraph tag fix: removes the proper spacing or the paragraph (p) tag.  To correct we set the top/bottom margin to 1em in the head of the document. Simple fix with little effect on other styling.  NOTE: 1em or 1.12 em http://www.w3.org/TR/CSS21/sample.html**/
p {
   margin: 1em 0;
   /** wonder if inherited font styles will fix outlook 2007 issue **/
   font-style: inherit;
} 
/** Fix for image spacing in Gmail and Hotmail.**/
img {
   display: block;
}

The good stuff:
This site was created to actually put pen to paper some of ours (and our friends') experiences when it comes to rendering html email. We thought, hey, how great would it be to use a cool new technology, like HTML5, to showcase some of the pitfalls of an older (rendering-wise) technology - in this case, email clients like Outlook, Gmail, Hotmail, etc.
We also thought, hey (which is for horses and sometimes cows), why don't we use Paul Irish's HTML5 Boilerplate (thanks for the inspiration Paul) as our starting point. And after a little love, peace, and html grease this site came to be.
