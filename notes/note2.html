The MailChimp Reset

A great reset from the good folks over at the email service provider. This snippet resets formatting issues with several clients including Outlook, Hotmail, and Gmail. It also includes a sweet fix for some smart phone clients where font sizes are increased to break your layout.

More info/Reference:

  • Mail Chimp Blog
  • MailChimp Blueprints Templates on Github

/*********** The MailChimp Reset from Fabio at MC More info and templates on Github: https://github.com/mailchimp/Email-Blueprints ***********/ /* Client-specific Styles */ #outlook a{padding:0;} /* Force Outlook to provide a "view in browser" button. */ body{width:100% !important;} /* Force Hotmail to display emails at full width */ body{-webkit-text-size-adjust:none;-ms-text-size-adjust:none;} /* Prevent Webkit platforms from changing default text sizes. SP: I added in the WinMobile one for good measure. */ /* Reset Styles */ body{margin:0; padding:0;} img{border:none; font-size:14px; font-weight:bold; height:auto; line-height:100%; outline:none; text-decoration:none; text-transform:capitalize;} /* When brought inline, this line of css fixes Gmail/Hotmail image whitespace issue*/ /** End Mail Chimp Reset **/ ...
