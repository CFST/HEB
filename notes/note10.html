BONUS: Fixing the Unwanted Yahoo Short Cut Links (Kind of)

Adaptation of Brian Thies link color fix to render the Yahoo Short Cuts invisible. You need to make the text color the same of the actual font color of your email and then set text-underline to none. NOTE: You then need to use the Responsys/Smith Harmon link color fix (tip #8) if you want to style your links anything other than blue.

.yshortcuts { color: black !important; text-decoration: none !important;} /* Body text color for the New Yahoo. This example sets the font to black. */ .yshortcuts a span { color: black !important; text-decoration: none !important; } /* Link text color for the New Yahoo. This example sets the font to black. */
