Yahoo Link Color Reset (1 of 2 fixes)

Yahoo link color reset (fix #1 of 2): This fix is from Brian Thies over at Campaign Monitor. It overwrites Yahoo mail's specific blue coloring on links. It, however, does not set color across email clients that ignore CSS in the HEAD tag. You will still have to set link color inline for gmail/others. NOTE: This example sets the link color red.

More info:

  • Two Solutions to Yahoo Turning Your Links Blue
a { color: red; text-decoration: underline;} /** Yahoo Classic and other clients that support styling within the HEAD tags **/ .yshortcuts { color: red !important; text-decoration: underline !important;} /* Body text color for the New Yahoo */ .yshortcuts a span { color: red !important; text-decoration: underline !important; } /* Link text color for the New Yahoo */
